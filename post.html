How I built my first website with Git and SSH

April 10, 2026

Today I built my first ever website from scratch. I had never written a single line of code before, but by the end of the day I had a live website on the internet.

Here's what I learned along the way.

What is Git?

Git is a version control system — it tracks every change you make to your code over time. Think of it like a save system in a video game. You can always go back to any previous save.

What is SSH?

SSH is a secure way to connect two computers. Instead of a password, it uses a pair of keys — a private key that stays on your computer, and a public key that you share with GitHub. They work together like a padlock and its key.

What I built

Using Git, SSH and GitHub Pages, I deployed this blog completely for free. Every time I want to update it, I just edit the files, commit and push — and it goes live in minutes.

Not bad for day one. 🌱
